# gunpla-stock-bot

> プレミアムバンダイのガンプラ在庫を自動でチェックし、再販・復活時に通知する個人用Bot。  
> **将来的には Playwright を活用したサイト E2E テストや、自動化アプリ開発へ発展**させることを目指します。

---

## 概要

- 指定したガンプラ商品の在庫ステータスを定期監視  
- 在庫「なし → あり/予約」を検知したら即通知（LINE / Discord / ntfy など）  
- オプションでログインセッションを使い、在庫あり時に **カート投入** まで自動化（決済はしません）

---

## 使い方（ざっくり）

1. Node.js を用意（v20+）  
2. `npm i` で依存を入れる  
3. `npx playwright codegen https://p-bandai.jp/` で一度ログインして `storageState` を保存  
4. `config/targets.json` に監視したい商品の URL を追加  
5. `npx ts-node src/main.ts` を実行（cron 登録すれば常時監視）

> 詳細は `仕様書 — gunpla-stock-bot (v0.2)` を参照。

---

## 目指す姿（Roadmap）

- **Phase 1**：在庫通知 + 任意のカート投入（個人用）  
- **Phase 2**：Playwright のシナリオを増やして **サイトの回帰テスト** を兼用（セレクタ安定化・モジュール化）  
- **Phase 3**：自動化パターンのテンプレ化 → **小さな自動化アプリ群**（入荷監視／変更検知／フォーム送信 etc.）に展開  
- **Phase 4**：Next.js ダッシュボードで可視化、通知・履歴・ヒートマップ（予約開始時刻の傾向）

---

## ライセンス / 注意

- 個人学習・検証目的に限定してください。過剰アクセスや自動購入は禁止。  
- サイト規約や法令に従い、適切なアクセス間隔で運用してください。
